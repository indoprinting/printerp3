# PrintERP 3

PrintERP 3 adalah aplikasi Enterprise Resource Planning (ERP) berbasis web milik CV. Indoprinting. Dibuat dari scratch sebagai optimalisasi dari PrintERP generasi pertama.

# Features
* SPA (Single Page Application)
* Division
  - Biller
  - Warehouse
* Finance
  - Bank Account
  - Bank Mutation
  - Bank Reconciliation
  - Expense
  - Income
  - Payment Validation (Mutasibank, QRIS)
* Human Resource
  - Customer
  - Customer Group
  - User
  - User Group
  - Supplier
* Inventory
  - Stock Adjustment
  - Stock Sync
  - Internal Use
  - Item Category
  - Item Mutation
  - Product
  - Stock Opname
  - Stock Transfer
  - Usage History
* Procurement
  - Purchase
